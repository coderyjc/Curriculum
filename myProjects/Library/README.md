## 图书馆管理系统

### 项目目录结构

|文件夹| 作用|
|----|---|
|Components| UI界面的组件，里面的子文件夹存放各种的自定义控件|
|Domain | 实体类，即POJO|
|imgs | UI中的背景图片|
|jar| 项目所使用的jar包 |
|UI| 程序使用的UI界面，都是Frame|
|Utils| 工具类，主要有数据库工具类和其他工具类|
